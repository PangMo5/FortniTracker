## FortniTracker
Fortnite Tracker for iOS with RxSwift and MVVM architecture

## App Features
- [x] DailyStore
- [x] UpcomingStore
- [ ] Item Detail
- [ ] UserSearch(Recent search...)
- [ ] UserTracking(UI Designing...)
- [ ] User, Item Bookmarks
- [ ] Share
- [x] Night Theme
- [ ] Add icon
